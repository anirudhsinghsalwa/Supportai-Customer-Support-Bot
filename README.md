## 🚀 Overview

SupportAI is a modern AI-powered customer support platform designed to automate customer interactions and provide instant intelligent responses.

The chatbot leverages Google's Gemini AI model to understand user queries, generate contextual responses, and maintain conversation history in a seamless ChatGPT-style interface.

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/supportai-customer-support-bot.git
cd supportai-customer-support-bot
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
SECRET_KEY=your-secret-key
DEBUG=True
GEMINI_API_KEY=your-gemini-api-key
```

### Run Database Migrations

```bash
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Launch Application

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
